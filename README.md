# JackrabbitAbundance
Extended N-mixture model to estimate black-tailed jackrabbit abundance from counts derived from repeated spotlight surveys.

Code created for: 
Perrig, Paula Leticia, John Clare, Zoe Amelia Bonerbo*, Leticia Camacho*, Jennyffer Cruz. In Review. Black-tailed jackrabbit abundance is closely linked to shrub cover in degraded sagebrush ecosystems. Journal of Mammalogy. 

Code includes Nimble model to estimate the relationship between black-tailed jackrabbit and three vegetation types sampled during 2022-2024. The model is an extended N-mixture model that jointly estimates detection, availability and abundance. 

Data are loaded as a workspace. The data comes from repeated counts of unmarked individuals collected by dependent, double observers, while driving at night along dirt roads. Visibility was truncated to 10 m-wide strip transects to ensure homogeneous detection, regardless of vegetation structure. 

Detailed data and model descriptions are described in detail in the manuscript.
